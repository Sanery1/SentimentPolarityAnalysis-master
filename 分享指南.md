## 给接收者的快速指南

### 第一步：安装 Python

确保已安装 Python 3.7 或更高版本：
- 下载地址：https://www.python.org/downloads/
- 安装时勾选 "Add Python to PATH"

验证安装：
```bash
python --version
```

### 第二步：安装依赖

在项目目录下运行：
```bash
pip install -r requirements.txt
```

等待所有包安装完成（大约需要 1-2 分钟）。

### 第三步：启动项目

**方式一：Web 界面（推荐）**
- Windows：双击 `启动Web服务.bat`
- Mac/Linux：运行 `python app.py`
- 然后在浏览器打开：http://localhost:5000

**方式二：命令行**
```bash
python run_demo.py
```

### 第四步：使用系统

Web 界面：
1. 在文本框输入句子（如："味道很好，服务也不错"）
2. 点击「🔍 分析情感」
3. 查看结果（正面/负面）

## 🐛 常见问题

### 问题1：找不到 Python 命令
**解决：**
- 重新安装 Python 并勾选 "Add to PATH"
- 或使用 `python3` 代替 `python`

### 问题2：pip 安装失败
**解决：**
```bash
# 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题3：启动后无法访问
**解决：**
- 检查防火墙设置
- 尝试访问 http://127.0.0.1:5000
- 确认终端显示 "Running on..."

### 问题4：中文显示乱码
**解决：**
- Windows 用户在终端运行：`chcp 65001`
- 使用 UTF-8 编码
